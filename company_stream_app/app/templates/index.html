Frequently Asked Questions (FAQ)

What is the Fab Stream Platform?

It is a secure, internal streaming service designed for our company to upload, manage, and view media content in a centralized environment.

How do I upload content?

Authorized users can use the upload dashboard to add new media files. All files are securely stored in our dedicated storage system.

What is the maximum file size allowed?

To ensure optimal performance, the current upload limit is set to 16 MB per file.

Is my data secure?

Yes. The application uses encrypted session management and a secure SQLite database backend to protect user information and metadata.

Which formats are supported?

We currently support standard video and audio formats. If you encounter an issue with a specific file type, please contact the administrator.
